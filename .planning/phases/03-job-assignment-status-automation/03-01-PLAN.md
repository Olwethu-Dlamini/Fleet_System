**A. Add `getDriverLoadStats` to Job.js** (new static method on the Job class):

Add a static method `getDriverLoadStats(tenantId, range = 'weekly')` that:
- Accepts range values: 'weekly' (7 days), 'monthly' (30 days), 'yearly' (365 days)
- Queries users LEFT JOIN job_technicians with the date range filter on `jt.assigned_at`
- Filters `WHERE u.role IN ('driver','technician') AND u.is_active = 1 AND u.tenant_id = ?`
- Returns array of objects: `{ id, full_name, job_count, rank, below_average }`
- `rank` is 1-indexed (rank 1 = lowest job_count = "Suggested" driver)
- `below_average` is boolean: true when `job_count < average` across all returned drivers
- If no drivers exist, return empty array
- Use the project's `db.query()` pattern (db = require('../config/database'))

The SQL query:
```sql
SELECT u.id, u.full_name, COUNT(jt.id) AS job_count
FROM users u
LEFT JOIN job_technicians jt ON jt.user_id = u.id {rangeClause}
WHERE u.role IN ('driver','technician')
  AND u.is_active = 1
  AND u.tenant_id = ?
GROUP BY u.id, u.full_name
ORDER BY job_count ASC, u.full_name ASC
```

Where `rangeClause` is:
- weekly: `AND jt.assigned_at >= DATE_SUB(NOW(), INTERVAL 7 DAY)`
- monthly: `AND jt.assigned_at >= DATE_SUB(NOW(), INTERVAL 30 DAY)`
- yearly: `AND jt.assigned_at >= DATE_SUB(NOW(), INTERVAL 365 DAY)`

Post-query: compute `avg = total / rows.length`, then map each row adding `rank: i + 1` and `below_average: row.job_count < avg`.

router.get('/driver-load', verifyToken, async (req, res) => {
  try {
    const range = req.query.range || 'weekly';
    if (!['weekly', 'monthly', 'yearly'].includes(range)) {
