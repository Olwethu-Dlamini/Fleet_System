1. Create `lib/models/time_extension.dart` with 4 classes:

   a. `TimeExtensionRequest`: id (int), jobId (int), requestedBy (int), durationMinutes (int), reason (String), status (String — pending/approved/denied), denialReason (String?), createdAt (DateTime). Factory `fromJson(Map<String, dynamic> json)`.

   b. `RescheduleOption`: id (int), requestId (int), type (String — push/swap/custom), label (String), changes (List<JobTimeChange>). Factory `fromJson` parsing changes from list of maps.

   c. `JobTimeChange`: jobId (int), jobNumber (String), currentStart (String?), currentEnd (String?), newStart (String), newEnd (String). Factory `fromJson`.

   d. `AffectedJob`: id (int), jobNumber (String), currentStart (String), currentEnd (String). Factory `fromJson`.

2. Create `lib/services/time_extension_service.dart`:
   - Import ApiService and AppConfig
   - `createRequest({required int jobId, required int durationMinutes, required String reason})` → POST to timeExtensionsEndpoint with { job_id, duration_minutes, reason }. Parse response into { request: TimeExtensionRequest, affectedJobs: List<AffectedJob>, suggestions: List<RescheduleOption> }.
   - `getActiveRequest(int jobId)` → GET to `${timeExtensionsEndpoint}/$jobId`. Return TimeExtensionRequest? and List<RescheduleOption>.
   - `approveRequest(int requestId, {int? suggestionId, List<Map<String, dynamic>>? customChanges})` → PATCH to `${timeExtensionsEndpoint}/$requestId/approve` with { suggestion_id, custom_changes }.
   - `denyRequest(int requestId, {String? reason})` → PATCH to `${timeExtensionsEndpoint}/$requestId/deny` with { reason }.

3. Create `lib/providers/time_extension_provider.dart`:
   - Extends ChangeNotifier
   - State: `TimeExtensionRequest? _activeRequest`, `List<RescheduleOption> _suggestions = []`, `List<AffectedJob> _affectedJobs = []`, `bool _loading = false`, `String? _error`
   - Public getters for all state fields
   - `submitRequest({required int jobId, required int durationMinutes, required String reason})` → async, sets loading, calls service.createRequest, sets _activeRequest/_suggestions/_affectedJobs, returns bool success. Catches errors and sets _error.
   - `loadActiveRequest(int jobId)` → fetches via service.getActiveRequest, sets state
   - `approveRequest(int requestId, {int? suggestionId, List<Map<String, dynamic>>? customChanges})` → calls service, clears _activeRequest on success
   - `denyRequest(int requestId, {String? reason})` → calls service, clears _activeRequest on success
   - `clearState()` → resets all fields, notifyListeners

4. Add to `app_config.dart`:
   `static const String timeExtensionsEndpoint = '/time-extensions';`
