1. Create `src/services/directionsService.js`:
   - `const logger = require('../config/logger').child({ service: 'directionsService' });`
   - `async function getDirections(originLat, originLng, destLat, destLng)`:
     - Call Google Routes API v2 via fetch:
       ```
       POST https://routes.googleapis.com/directions/v2:computeRoutes
       Headers:
         Content-Type: application/json
         X-Goog-Api-Key: process.env.GOOGLE_MAPS_API_KEY
         X-Goog-FieldMask: routes.duration,routes.distanceMeters,routes.polyline.encodedPolyline
       Body:
         origin: { location: { latLng: { latitude: originLat, longitude: originLng } } }
         destination: { location: { latLng: { latitude: destLat, longitude: destLng } } }
         travelMode: DRIVE
       ```
     - Parse response: extract `routes[0].polyline.encodedPolyline`, `routes[0].duration` (string like "1250s"), `routes[0].distanceMeters` (number)
     - Convert duration: parse seconds from string, format as "X hr Y min" or "Y min"
     - Convert distance: `(distanceMeters / 1000).toFixed(1)` + " km"
     - Return: `{ encoded_polyline, duration_text, duration_seconds, distance_text, distance_meters }`
     - On error: log and throw
   - Export: `{ getDirections }`

2. Add to `src/routes/gps.js` — **GET /directions** endpoint:
   - `const { verifyToken } = require('../middleware/auth');`
   - `const DirectionsService = require('../services/directionsService');`
   - `const db = require('../config/database');`
   - Route: `router.get('/directions', verifyToken, async (req, res) => { ... })`
   - Query param: `job_id` (required)
   - Fetch job from DB: `SELECT destination_lat, destination_lng FROM jobs WHERE id = ? AND tenant_id = ?`
   - If job not found or no destination coords, return 404
   - For origin: use query params `origin_lat` and `origin_lng` if provided. If not provided, return the destination coords only with a note that origin is needed for full directions.
   - Call `DirectionsService.getDirections(originLat, originLng, destLat, destLng)`
   - Return 200: `{ success: true, directions: { encoded_polyline, duration_text, distance_text, destination_lat, destination_lng } }`
   - Catch errors: return 500 with message (do NOT expose API key errors to client)

If gps.js does not exist yet, create it with:
- `const express = require('express');`
- `const router = express.Router();`
- The directions GET route
- `module.exports = router;`
And ensure it is registered in routes/index.js (add `const gpsRoutes = require('./gps'); router.use('/gps', gpsRoutes);` if not already there).
