    **Create `lib/providers/gps_provider.dart`:**
    A ChangeNotifier managing GPS consent state and the location posting timer.

    Fields:
    - `bool _consentChecked = false` ã whether we've checked the DB yet
    - `bool _consentGranted = false` ã whether user has consented
    - `bool _gpsEnabled = false` ã whether GPS is currently active (consent can be granted but disabled)
    - `bool _loading = false`
    - `Timer? _locationTimer`

    Methods:
    - `Future<void> checkConsent()` ã calls `GpsService.getConsent()`. If response is null (no record), sets `_consentChecked = true, _consentGranted = false`. If response has `gps_enabled`, sets both `_consentGranted = true` and `_gpsEnabled = response['gps_enabled']`. Also reads `SharedPreferences.getBool('gps_consent_given')` as fast cache ã if true, skip API call initially but still refresh from API in background.
    - `Future<bool> grantConsent()` ã calls `GpsService.grantConsent()`. On success, sets `_consentGranted = true, _gpsEnabled = true`. Writes `SharedPreferences.setBool('gps_consent_given', true)`. Calls `startLocationTimer()`. Returns success bool.
    - `Future<bool> toggleGps(bool enabled)` ã calls `GpsService.updateConsent(enabled)`. On success, sets `_gpsEnabled = enabled`. If disabled, calls `stopLocationTimer()`. If enabled, calls `startLocationTimer()`. Returns success bool.
    - `void startLocationTimer()` ã only starts if `_gpsEnabled == true` and timer is null. Creates `Timer.periodic(Duration(seconds: 30))` that: checks `_isWithinWorkingHours()` (hour >= 6 && hour < 20 using DateTime.now()), if within hours calls `Geolocator.getCurrentPosition(locationSettings: LocationSettings(accuracy: LocationAccuracy.high))` then `GpsService.postLocation(lat: pos.latitude, lng: pos.longitude, accuracyM: pos.accuracy)`. All errors caught and logged (non-fatal).
    - `void stopLocationTimer()` ã cancels timer, sets to null.
    - `bool _isWithinWorkingHours()` ã `final now = DateTime.now(); return now.hour >= 6 && now.hour < 20;`
    - `void dispose()` ã stops timer.

    Getters: `consentChecked`, `consentGranted`, `gpsEnabled`, `isLoading`, `needsConsent` (= consentChecked && !consentGranted).

    Import: `dart:async`, `package:flutter/material.dart`, `package:geolocator/geolocator.dart`, `package:shared_preferences/shared_preferences.dart`, gps_service.dart.

       ```dart
       // After auth.status == authenticated:
       final gps = context.watch<GpsProvider>();

       // Only check consent for driver/technician roles
       if (auth.isTechnician) {
         // Trigger consent check if not done yet
         if (!gps.consentChecked) {
           // Use Future.microtask to avoid calling during build
           Future.microtask(() => gps.checkConsent());
           // Show loading while checking
           return const Scaffold(body: Center(child: CircularProgressIndicator()));
         }
         // If consent not yet granted, show consent screen
         if (gps.needsConsent) {
           return const GpsConsentScreen();
         }
         // Consent resolved ã start location timer if enabled
         if (gps.gpsEnabled && !gps.isTimerRunning) {
           Future.microtask(() => gps.startLocationTimer());
         }
       }

       return const MainApp();
       ```

       Note: Keep the existing unauthenticated and unknown checks exactly as they are. The consent gate only applies to technician/driver roles. Admin and scheduler skip straight to MainApp.
